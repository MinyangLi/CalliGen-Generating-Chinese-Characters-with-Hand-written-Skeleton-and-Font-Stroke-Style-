# CalliGen-Generating-Chinese-Characters-with-Hand-written-Skeleton-and-Font-Stroke-Style
## Generation
### Requirements
```bash
# Install dependencies
pip install -r requirements_full.txt
```
### Data preparation
#### 1. Prepare your TrueType font file.
#### 2. Specify the path of your TrueType font file, the directory of the character images, and the skeleton images in "generation_better_data.py".
#### 3. Run the following command
```bash
python generate_better_data.py
```
